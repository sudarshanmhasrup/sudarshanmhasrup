About

Started with curiosity,
stayed for Kotlin.

Hi — I'm Sudarshan, a software developer from India who fell in love with Kotlin and never looked back. What began as curiosity about how software works has grown into a genuine obsession with writing, testing, debugging, and shipping code that solves real problems.

I've dedicated myself to building native and cross-platform solutions using Kotlin and Compose Multiplatform, targeting Android, iOS, desktop, and web from a single codebase. One language. One UI toolkit. Every platform.

Beyond code, I produce music, follow cricket with an unhealthy passion (MS Dhoni, always), and believe the only real way to learn a technology is to build something with it — no shortcuts.
